About Us






The Scottish Football Quiz was created for educational purposes as a project for the Hons BCS Web Design & Development course at Napier University based in Edinburgh. The learning objectives of the project were to demontstrate ability in authoring tools and markup languages, client side and server side programming as well as applying appropriate web page development methodology.


All information, resources and images are not intended for commercial use.
